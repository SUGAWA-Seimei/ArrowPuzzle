# ArrowPuzzle  
矢印をタップして外してくゲームです。  
短編のマス数を設定して「新規」ボタンを押すと新しいゲームをスタートできます。 

[https://sugawa-seimei.github.io/ArrowPuzzle/](https://sugawa-seimei.github.io/ArrowPuzzle/)

---
## 遊び方 
- 矢印の進行方向に障害物がなければ、タップすると消えます。  
- 進行方向に別の矢印があると、その矢印は消せません。
- 消せない矢印をタップするとライフを一つ失います。
- ライフを全て失ったらゲームオーバーです。  
- すべての矢印を盤面から消せばクリアです。
